Edit the `setup` script. Replace lines 7-9 (the current `if command -v zsh` + `sudo chsh` block):

```bash
# Current (lines 7-9):
if command -v zsh >/dev/null; then
  sudo chsh -s "$(command -v zsh)" "$USER"
fi
```

With the container-guarded version:

```bash
# Only change default shell on host machines — skip in DevPod/Codespaces/devcontainers
# where sudo or /etc/passwd may be unavailable
if [[ -z "$REMOTE_CONTAINERS" && -z "$CODESPACES" && -z "$DEVCONTAINER_TYPE" ]]; then
  if command -v zsh >/dev/null; then
    sudo chsh -s "$(command -v zsh)" "$USER"
  fi
fi
```

This reuses the exact same container detection pattern from `dot_zshrc` line 8.

DO NOT modify any other part of the setup script (chezmoi install, pure prompt clone, alacritty themes section).
